# 电商用户行为数据分析项目



## &#x20;项目背景

本项目基于真实电商平台用户数据，对用户消费行为和流失情况进行分析，帮助理解用户特征并为运营策略提供支持。





### 项目亮点

* 使用SQL进行数据提取与聚合分析
* 使用Python完成数据清洗与分析
* 构建完整的数据分析流程（清洗 → 分析 → 可视化）





### 技术栈

1. Python（pandas, matplotlib）
2. SQL（MySQL）
3. 数据可视化





### 分析内容



##### 商品分析

* 商品类别分布均匀，无明显偏向



##### 用户消费分析

* 用户存在消费集中在中等区间
* 存在少量高消费用户



##### 用户流失率分析

* 流失率约为20%
* 流失用户与消费金额关系不明显
* 年轻用户流失率稍高





### 项目成果

1. 完成从数据清洗 → 分析 → 可视化的完整流程
2. 使用SQL进行数据提取与聚合分析
3. 输出可视化图表支持分析结论

业务结论以及建议

1. 用户流失率约为20%，主要集中在低消费人群（平均消费约2700元）
   → 建议针对低消费用户推出优惠券或满减活动，提高留存

2. 20岁年龄段流失率最高（约23%）
   → 建议针对年轻用户设计会员体系或社交裂变活动

3. 女性用户流失略高于男性
   → 可针对女性用户优化商品推荐或营销策略

4. 高消费用户流失较少
   → 应重点维护高价值用户（VIP体系、专属优惠等）

总的来说通过“低消费拉新 + 高消费留存”，优化用户结构，提高整体营收
