# 电商用户行为数据分析项目



## &#x20;项目背景

本项目基于真实电商平台用户数据，对用户消费行为和流失情况进行分析，帮助理解用户特征并为运营策略提供支持。





### 项目亮点

* 使用SQL进行数据提取与聚合分析
* 使用Python完成数据清洗与分析
* 构建完整的数据分析流程（清洗 → 分析 → 可视化）





### 技术栈

1. Python（pandas, matplotlib）
2. SQL（MySQL）
3. 数据可视化





### 分析内容



##### 商品分析

* 商品类别分布均匀，无明显偏向



##### 用户消费分析

* 用户存在消费集中在中等区间
* 存在少量高消费用户



##### 用户流失率分析

* 流失率约为20%
* 流失用户与消费金额关系不明显
* 年轻用户流失率稍高





### 项目成果

1. 完成从数据清洗 → 分析 → 可视化的完整流程
2. 使用SQL进行数据提取与聚合分析
3. 输出可视化图表支持分析结论





### &#x20;项目结构

1. ecommerce\_analysis.py → Python分析代码
2. ecommerce.sql → SQL分析脚本
3. user\_spending.png → 用户消费分布图
4. churn\_vs\_spending.png
